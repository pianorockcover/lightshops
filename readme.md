# LightShops - E-commerce HTML theme

[DEMO](https://pianorockcover.github.io/lightshops/)

Multipurpose e-commerce html template. Written in pug and LESS. Free for any use.

## Build for production

```bash
yarn build
```

## Development

```bash
yarn dev
```
